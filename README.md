# Minimal React + Tailwind CSS 4 + shadcn/ui Template

A minimal, clean template with React, Tailwind CSS 4, and shadcn/ui components.

## Features

- ⚡ Vite for fast development
- 🎨 Tailwind CSS 4 for styling
- 🧩 shadcn/ui components
- 📦 Minimal dependencies
- 🔧 ESLint for code quality

## Getting Started

```bash
npm install
npm run dev
```

## Adding shadcn/ui Components

```bash
npx shadcn@latest add [component-name]
```

## Tech Stack

- React 19
- Tailwind CSS 4
- shadcn/ui
- Vite
- ESLint
